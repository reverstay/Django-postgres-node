How to Embed Apps

The Basic Call Structure

Below is the basic template used to embed an app on any site.

<iframe src="{{HTTP_HOST}}embed/APP/TRADE_FLOW/COUNTRY1/COUNTRY2/PRODUCT/YEAR/"></iframe>

Which you will notice has the following parameters: COUNTRY / COUNTRY / PRODUCT / YEAR, a byproduct of the bilateral nature of the dataset from which the apps are being built.

URI Options

stacked Build a stacked area chart of trade data over an interval of years.
